# TongDaRailway Mod 通达铁路

这是一个添加自动生成机械动力铁路的模组

**此模组只维护1.21.1 NeoForge版本，无其他版本适配计划！**

**模组更新修改地形生成时，使用老版本模组的世界在删除已生成的非必要区块前，请勿更新新版本模组**

## 1.1.0 更新路线生成算法

* 路线图大致生成步骤：
* 使用低差异序列在区域内随机均匀的选择站点
* 去除不合法的选点（如海洋上的点）
* 将站点连接成为德劳内三角网
* 删边，使得圆周上的相邻边在特定角度以上
* 删边，使得节点的最大度为4。根据节点度数分配对应出口的车站
* 返回节点和它所连接的所有节点
* 通过匈牙利匹配为每个车站出口匹配一条边
* 根据匹配结果生成连接
* 现在只有某些区域会生成铁路。有些区域则有概率不生成铁路
